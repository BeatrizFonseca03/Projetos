# Classificação de Cancro do Pulmão com Imagens CT

Projeto académico desenvolvido com o objetivo de aplicar técnicas de Machine Learning à classificação de nódulos pulmonares a partir de imagens de Tomografia Computorizada (CT).

Foram utilizados dados do dataset LIDC-IDRI, contendo exames anotados por radiologistas. O trabalho incluiu as seguintes etapas:

- Pré- processamento de imagens médicas (normalização e tratamento de dados)
- Análise exploratória
- Extração e seleção de features radiómicas
- Implementação de modelos de Machine Learning: Random Forest, SVM, XGBoost
- Avaliação com métricas de desempenho

O projeto pretende explorar abordagens automáticas de apoio ao diagnóstico médico e consolidar competências em tratamento de dados, modelação e avaliação de modelos.

Este repositório contém os notebooks e código desenvolvidos durante o trabalho.
