# Classificação de Cancro do Pulmão com Imagens CT

Projeto académico desenvolvido com o objetivo de aplicar técnicas de Machine Learning à classificação de nódulos pulmonares a partir de imagens de Tomografia Computorizada (CT).

Foram utilizadas imagens do dataset LIDC-IDRI e realizadas etapas de pré-processamento, análise exploratória e treino de modelos para avaliação de desempenho na tarefa de classificação. O projeto pretende explorar abordagens automáticas de apoio ao diagnóstico médico e consolidar competências em tratamento de dados, modelação e avaliação de modelos.

Este repositório contém os notebooks e código desenvolvidos durante o trabalho.
