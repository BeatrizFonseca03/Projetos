# Previsão e Otimização de Portfólio de Ações

Projeto académico focado na aplicação de técnicas de Machine Learning à previsão de séries temporais financeiras e à otimização de portfólios de investimento.

O trabalho utiliza dados históricos do índice S&P500 para prever o comportamento de ações e aplicar métodos de otimização na seleção de ativos, com o objetivo de maximizar retorno ou reduzir risco. Foram realizadas etapas de pré-processamento, análise exploratória, treino e avaliação de modelos, bem como simulação de estratégias de investimento.

Este repositório contém os notebooks e código desenvolvidos durante o projeto.
