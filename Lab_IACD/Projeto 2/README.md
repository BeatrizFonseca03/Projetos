# Previsão e Otimização de Portfólio de Ações

Projeto académico focado na aplicação de técnicas de Machine Learning à previsão de séries temporais financeiras e à otimização de portfólios de investimento, maximizando o retorno ajustado ao risco.

O trabalho utiliza dados históricos (2010-2024) do índice S&P500 para prever o comportamento de ações e aplicar métodos de otimização na seleção de ativos, com o objetivo de maximizar retorno ou reduzir risco. 
Foram realizadas as seguintes etapas:

- Pré-processamento e criação de features temporais
- Aplicação dos algoritmos AdaBoost, Random Forest LightGBM e LSTM para modelar os preços das ações
- Avaliação dos modelos usando métricas de erro (RMSE/MAE) e posteriormente integrados em simulações de estratégias de investimento, recorrendo ao Monte Carlo, em período futuro (2024)

Este repositório contém os notebooks e código desenvolvidos durante o projeto.
