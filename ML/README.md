# Classificação de Sons Urbanos com Deep Learning

Este projeto tem como objetivo a criação de classificadores de deep learning capazes de identificar automaticamente sons urbanos a partir de excertos de áudio.

O trabalho utiliza o dataset **UrbanSound8K**, composto por amostras de áudio com duração de até 4 segundos, distribuídas por 10 classes distintas de sons urbanos. Para resolver a tarefa de classificação, foram implementados e avaliados dois modelos de redes neuronais: **Multilayer Perceptron (MLP)** e **Convolutional Neural Network (CNN)**.

O projeto inclui todas as etapas principais de um pipeline de aprendizagem automática: pré-processamento e preparação dos dados áudio, definição das arquiteturas dos modelos, configuração das estratégias de treino e avaliação de desempenho. A avaliação foi feita através de cross validation em 10 folds, permitindo obter métricas de desempenho e comparar a eficácia das diferentes abordagens.
